Cara Penggunaan Git

1) Search github & daftar

2) Cari new repository dan buat namenya kemudan scroll ke bawah klik publik dan add a readme file

3) Buat folder baru kemudian Install git=scm setelah ke install masukkan codingan dengan format folder yang udah dibuat seperti gambar berikut salin code yang udah di buat di git=scm dan masukan ke codingan tsb juga

4) Setelah file masuk ke folder lalu download visual studio code dan buka folder tsb dengan visual studio code

5) Buat file baru di VSC tsb. kemudian file baru klik html5 nanti keluar codingan baru dan anda bisa lanjutkan codingan tsb

6) Setelah itu save
